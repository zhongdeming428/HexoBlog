我安装的是 Tomcat 7 版本，在 Ubuntu 18.04 上运行，IDE 为 Intellij IDEA。

首先创建一个 Java Web 项目，进入你的 IDEA，然后点击 Create New Project。如下图所示：

选择完毕之后点击下一步。
